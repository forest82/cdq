优秀的人总是很有情怀的

Posted on 三月 11, 2018 本文总阅读量次

悄悄话

写文章一个比较尴尬的事，本来就没啥墨水，好不容易找到自己的一个角度看这个问题，然后准备搜索各种解释，提炼出自己的观点。在这之中，发现一篇不错的跟自己立意相同的文章，彻底沦陷，然后动摇了，自己真的要写？写的这么好的已经有了，怎么办啊？转载？？很好，可是脱离了自己刚开始写文章原创的想法

这可能是很多人提笔又放下的原因，刚开始都想做的尽善尽美，可是也应该清楚，什么事都不是一蹴而就的，一口也不成一个胖子，想想自己要原创的最初的想法。。。只能结合百家之长，写出自己的风格和见地吧

关键词 前端 名字 起源

历史成为历史很大一部分原因在于其没有温度，只是书本上的文字，眼前背了一遍又一遍的出现的文字组合，但是如果你深挖其背后的故事，它的温度就出来了。人是有温度的，也更愿向温度靠近

你的名字

在代码世界里，我们接触的很多东西都是英文的，有时候不懂就查下它的意思，最多看下这个框架或名字的官方定义的用途，对这个东西的探索也就到此结束了，一段时间之后，对这匆匆理解的一个东西，或许又要重新来一遍“认识”。

好了，不罗嗦了，分享我看到的比较好玩的命名来源

polyfill

A polyfill, or polyfiller, is a piece of code (or plugin) that provides the technology that you, the developer, expect the browser to provide natively. Flattening the API landscape if you will.（一段代码或插件，可以让开发人员使用应有的技术，就像浏览器原生提供该功能一样。换句话说，它能帮你抹平API之墙。）What is a Polyfill?

这种填料在美国叫Spackling Paste（Spackle是美国抹墙粉的一个品牌），当然在我们中国一般叫“腻子”或“填泥”（对应的英文单词是putty和filler）。
为什么用了英国的名字呢，作者是英国人 转载：李松峰：一袋“腻子粉”的故事

Babel

语法解析器、丰富的插件平台

Babel 是一个 JavaScript 的语法解析器，主要功能是将 es6 的代码转换成 es5 的代码，实现浏览器的兼容，这也正是 Bable 的前身 6to5 的由来。但是考虑到以后也将兼容别的ECMAScript的规范，这个命名并没有通用性，经过大量讨论之后，最终选择Babel

故事的开始，由英国作家道格拉斯·亚当斯所写的一系列科幻小说《银河系漫游指南》（The Hitchhiker’s Guide to the Galaxy）中的虚构鱼形机械生物巴别鱼(Babel Fish)，而“巴别鱼”的名称又来自《圣经·创世纪》关人类在被上帝搞乱了他们的语言时所兴建的建筑物巴别塔(Tower of Babel)。

很久以前，人类的语言是共通的。巴比伦人想造一座通天塔，于是触怒上帝。上帝毁了塔，并且创造了不同的语言，使人类彼此无法沟通，以后就再也不能齐心协力造通天塔了。从此巴别塔就和语言有了关系。而 Babel 的功能就是把 ES6 转换成 ES5，也有统一语言的意义。
转载：瞬光知乎

结语：所有东西都是有温度的，鼓励大家慢慢学会看第一手的资料(东西的源头)，习惯了之后，思想沉淀的深度也会不一样。

有输入也要有输出，热爱阅读，热爱写作，坚持原创，发扬原创，记录每一步留下的脚印~
